파이 게임
제가 이전에 만들어 보았던 프로젝트를 모아둔 페이지 입니다.
현재는 너무 터무니 없는 완성도 이지만 접차 공부를 하여 추후에는 더 낳은 프로젝트를 늘려나갈 생각입니다.
  1. 벚꽃 카드게임 서비스 : 2021년도 벚꽃 행사 때 실제로 배포했던 서비스로 js를 이용해 만들어 보았습니다.
  2. 얼굴형 테스트 : Python을 이용해 Crawling을 진행 해 사진 data를 수집하였고, 구글이 만든 teacherble machine api를 이용해 만들어 보았습니다. 또한 댓글 서비스도 달아 보았습니다.
  3. 파이 게임 : Python 라이브러리인 Pygame을 이용해 만든 게임이다. 유투브에서 강의를 진행하는 나도 코딩님의 강의를 보면서 클론코딩 하였다.
